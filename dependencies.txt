# ------
# 1. OpenCV Version Compatibility:
#    - OpenCV 4.7+ uses cv2.aruco.ArucoDetector() class (new API)
#    - OpenCV < 4.7 uses cv2.aruco.detectMarkers() function (legacy API)
#    - This code automatically detects and uses the appropriate method
#    - Raspberry Pi OS typically ships with OpenCV < 4.7
#
# 2. On Raspberry Pi OS, you may prefer to use system packages:
#    sudo apt install python3-opencv python3-numpy python3-pil
#    Then only install: pip3 install picamera2 websockets requests

# 3. If you get OpenCV compilation errors, try:
#    pip3 install opencv-python-headless
#
# 4. For older Raspberry Pi models or specific compatibility, use:
#    pip3 install opencv-python==4.5.5.64
#
# 5. picamera2 requires libcamera support:

#    sudo usermod -a -G video $USER
